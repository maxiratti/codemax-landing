Prompt utilizado

Prompt completo para la generación de las Landing Pages

Este documento contiene el prompt utilizado para solicitar a distintos agentes de Inteligencia Artificial el desarrollo de dos Landing Pages para una academia de programación, manteniendo exactamente la misma estructura y contenido. 


Eres un Senior Full-Stack Developer con más de 15 años de experiencia
en desarrollo de software y diseño de interfaces modernas.

El objetivo es desarrollar un proyecto FrontEnd profesional,
completamente responsive, accesible y optimizado,
utilizando únicamente HTML5,
CSS3
y JavaScript Vanilla.

Todo el código debe ser limpio,
modular,
bien organizado
y seguir las mejores prácticas.

────────────────────────────────────────────

OBJETIVO

El proyecto estará compuesto por cuatro páginas:

• index.html
• prompt.html
• landing-ai-1.html
• landing-ai-2.html

Además deberá existir una estructura organizada
para CSS,
JavaScript,
assets,
imágenes e iconos.
